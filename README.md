# Collection Vault

A personal collection management app to track owned items, wishlists, photos, and usage over time.

## Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS
- Clerk Authentication
- Cloudinary Image Hosting
- Prisma + PostgreSQL (planned)

## Features (Planned)
- Secure authentication
- Create, edit, and delete collection items
- Photo uploads for items
- Tagging and filtering
- Usage tracking and analytics (cost-per-use, frequency)
- Wishlist management

## Getting Started
```bash
npm install
npm run dev
